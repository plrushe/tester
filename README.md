# Windows XP Desktop Website Concept

This project is a lightweight, interactive website shell that mimics a Windows XP desktop.

## Chosen technologies

- **HTML5** for semantic desktop/window/taskbar structure.
- **CSS3** for XP-inspired visual styling (wallpaper, window chrome, taskbar, icon states).
- **Vanilla JavaScript (ES6)** for interactivity:
  - open/close desktop windows,
  - simple taskbar window toggles,
  - focus ordering (`z-index`) for active windows,
  - live taskbar clock.

This stack is ideal for this concept because it is fast, dependency-free, and easy to host statically.

## Run locally

Open `index.html` directly in a browser, or serve with a static server.
